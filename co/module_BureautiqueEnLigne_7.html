Conclusion

Récapitulons :

  1. Vous écrivez votre mail directement dans Gmail ;

  2. Vous précisez dans votre mail les champs de fusion <<VotreChamp>> auxquels faire appel. Ces champs de fusion correspondent aux étiquettes des colonnes de votre feuille Google Sheets ;

  3. Vous sauvegardez le brouillon dans Gmail ;

  4. Vous compilez les informations dans votre feuille de calcul ;

  5. Vous envoyez vos emails personnalisés depuis votre feuille de calcul ;

  6. Vous suivez l'ouverture des emails par vos destinataires
