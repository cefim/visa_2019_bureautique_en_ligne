GMAIL avec Google Drive

Vous souhaitez réaliser simplement un publipostage dans Gmail, comme il est possible de le faire avec Word et Excel?

L'avantage de cette technique est la personnalisation des correspondances. Cela permet d'être plus efficace en entreprise et un engagement plus fort de la part des destinataires.

Cela est possible grâce à un module complémentaire de GOOGLE SHEETS, “Yet Another Mail Merge” YAMM, à installer.

Beaucoup plus simple que Mailchimp par exemple.

Vous devez donc partir de votre base de données qui sera sur Google sheets (coordonnées des destinataires, les colonnes doivent être nommées et cela servira de champ pour le publipostage.

Trouver Google Sheets dans les applications

Cliquer ensuite sur le + pour créer une feuille de calcul

Cliquer sur modules complémentaires, puis télécharger des modules

Chercher Yet Another Mail Merge

Télécharger Yet Another Mail Merge

Via la feuille de calcul, on va commencer le publipostage.

Pour ce faire, cliquer sur modules complémentaires puis Yet Another Mail Merge

Ensuite, on choisit dans les menus que le destinataire soit désigné par son email (si la colonne s'appelle E-mail., attention, il faudra écrire exactement “E-mail” et non “email”.)

On peut envoyer gratuitement 50 emails par jour (par 24h).

Ensuite dans un GMAIL, nouveau message, on écrit le contenu de son courrier par exemple :

“Bonjour $%Prénom% $%Nom%

Je vous convie à un cocktail le 16 avril prochain, votre numéro de téléphone est-il toujours le $%Tél.% ?

Bonne journée..... “

On peut également mettre un champ dans l'objet: exemple: “Courrier pour Mme $%Nom%”

On sauvegarde son mail dans les brouillons

Ensuite pas besoin de rentrer les adresses dans l'email puisque cela va être fait depuis Sheets

On a donc lancé le publipostage depuis Modules complémentaires/YET another mail merge/démarrer le publipostage/ remplir les paramètres

et vous pouvez également faire un mail test qui ne vous sera envoyé qu'à vous

Voici le résultat quand l'emailing est fini dans le tableau (ci-dessous) où vous pourrez suivre l'ouverture et la réception ou pas des emails dans la prochaine colonne vide.

NB : Si vous souhaitez faire une nouvelle campagne d'emailing, vous devez supprimer la colonne, pour qu'elle soit à nouveau utilisable.
